# ShareEnv Skills Catalog

This project keeps reusable AI engineering skills under `.agents/skills`.

## Installed Baseline Skills

- `find-skills`: discover and install additional skills.
- `frontend-design`: high-quality frontend design guidance.
- `vercel-react-best-practices`: React and Next.js performance and correctness patterns.
- `vercel-react-view-transitions`: animation and transition patterns.

## Project-Specific Skill

- `shareenv-crypto-engineering`
  - protocol-first cryptography workflow
  - zero-knowledge API guardrails
  - threat-model-driven change checklist

## How to Use

1. Load the relevant `SKILL.md` before implementation.
2. Follow reference docs in the skill `references/` folder.
3. Update protocol and threat docs whenever envelope semantics change.
4. Preserve zero-knowledge invariants in all API and persistence changes.
